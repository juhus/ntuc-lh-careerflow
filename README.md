# CareerFlow

End-to-End Career Placement Programme Management Platform.

Submission for **IMDA Open Innovation Platform · Call 28** — Challenge owner: **NTUC LearningHub**.

## Links
- **Prototype:** [index.html](./index.html)
- **Deck:** [deck.html](./deck.html)

## Four Modules
1. Smart Enrolment — AI-powered trainee assessment and course matching
2. Trainee Tracker — Real-time pipeline from enrolment to verified employment
3. Placement Engine — Skills-to-job matching with at-risk alerts
4. Analytics & Compliance — KPI dashboard, SSG reporting, PDPA audit trails
